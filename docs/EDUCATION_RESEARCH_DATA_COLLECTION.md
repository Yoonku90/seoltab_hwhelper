# 교육 연구 데이터 수집 가이드

## 🎯 목표

국영수사과 각 과목별로 학습 활동과 점수 향상의 상관관계를 보여주는 연구 데이터를 수집하여 초기 로드맵에 활용

## 📊 데이터 수집 전략

### 방법 1: 학술 논문 데이터베이스 활용

#### 주요 데이터베이스:
1. **RISS (한국교육학술정보원)**
   - URL: https://www.riss.kr
   - 한국어 교육 연구 논문 검색
   - 검색어: "어휘 학습", "문제 풀이", "성적 향상", "상관관계"

2. **DBpia (한국학술정보)**
   - URL: https://www.dbpia.co.kr
   - 교육학, 심리학 논문 검색

3. **Google Scholar**
   - URL: https://scholar.google.com
   - 국제 학술 논문 검색
   - 검색어: "vocabulary learning", "test score improvement", "correlation"

4. **ERIC (미국 교육 연구 정보 센터)**
   - URL: https://eric.ed.gov
   - 영어권 교육 연구 논문

#### 검색 키워드:
- **영어**: "vocabulary size", "English proficiency", "correlation", "effect size"
- **수학**: "problem solving", "math achievement", "practice effect"
- **국어**: "어휘력", "독해력", "성적 향상", "상관관계"
- **일반**: "deliberate practice", "spaced repetition", "learning time"

### 방법 2: 교육부/교육연구원 통계 활용

#### 주요 기관:
1. **한국교육개발원 (KEDI)**
   - 교육 통계 연보
   - 학업성취도 평가 결과

2. **한국교육과정평가원 (KICE)**
   - 국가 수준 학업성취도 평가
   - 대학수학능력시험 통계

3. **교육부**
   - 교육 통계 연보
   - 학생 성취도 조사

### 방법 3: 메타 분석(Meta-analysis) 논문 활용 ⭐

#### 장점:
- 여러 연구를 종합하여 더 신뢰성 있는 결과
- Effect size (효과 크기) 제공
- 일반화 가능한 결론

#### 검색 키워드:
- "meta-analysis vocabulary learning"
- "meta-analysis math problem solving"
- "meta-analysis study time achievement"

## 🔍 실제 데이터 수집 워크플로우

### Step 1: 논문 검색
1. Google Scholar 접속
2. 검색어 입력 (예: "vocabulary size English proficiency")
3. 최근 10년 내 논문, 메타 분석 우선 선택
4. PDF 다운로드 또는 요약 확인

### Step 2: 데이터 추출
1. 논문에서 다음 정보 추출:
   - 상관계수 (Correlation coefficient)
   - Effect size (Cohen's d, r 등)
   - 표본 크기 (Sample size)
   - 통계적 유의성 (p-value)

### Step 3: 데이터 정리
```typescript
interface ResearchData {
  subject: string;
  activity: string;
  targetCount: number;
  scoreImprovement: number; // 평균 점수 상승
  correlation?: number; // 상관계수
  effectSize?: number; // 효과 크기
  sampleSize: number; // 표본 크기
  source: string; // 논문 제목, 저자, 연도
  sourceUrl?: string; // 논문 URL
  confidence: 'high' | 'medium' | 'low';
}
```

### Step 4: 데이터베이스 구축
```typescript
const researchData: ResearchData[] = [
  {
    subject: 'english',
    activity: 'vocabulary_study',
    targetCount: 1000,
    scoreImprovement: 10.5,
    correlation: 0.68,
    effectSize: 0.75,
    sampleSize: 523,
    source: 'Nation (2001), "Learning Vocabulary in Another Language"',
    confidence: 'high'
  },
  // ...
];
```

## ⚠️ 주의사항

### 1. 데이터 출처 명시
- 모든 추정치에 출처 명시
- "일반적인 교육 경험치" 또는 "연구 데이터" 구분

### 2. 신뢰도 표시
- High: 메타 분석, 대규모 연구
- Medium: 단일 연구, 중간 규모
- Low: 추정치, 소규모 연구

### 3. 업데이트 계획
- 실제 데이터 수집 후 점진적 교체
- 정기적으로 최신 연구 검토
