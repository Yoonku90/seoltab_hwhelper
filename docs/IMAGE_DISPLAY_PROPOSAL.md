# 이미지 표시 방식 개선 제안

## 🔍 현재 상태

### 현재 구조:
1. **헤더 (chatHeader)**: 작은 썸네일 이미지 (40x40px) ✅ 이미 존재
   - 위치: 오른쪽 (프로필 사진과 이름 왼쪽)
   - 현재 표시: 이미지 썸네일 + 📄 아이콘
   - 클릭 시: 확대 모달 표시 ✅

2. **채팅 입력창 근처 (pageImageNearInput)**: 큰 이미지 (max-height: 150px) ❌ 문제
   - 위치: 채팅 로그 아래, 입력창 위
   - 현재 표시: 큰 이미지 + "필요시 클릭해서 보세요! (업로드한 이미지) 👆"
   - 문제: **학습 내용을 가릴 수 있음**

### 현재 레이아웃:
```
[📷 헤더]
  [프로필 사진] [선생님 이름] [📷 작은 썸네일 (40x40)] [복습 수업]

[채팅 로그]
  - 메시지들...
  - 메시지들...
  
[📷 큰 이미지 - 문제!]
  - 큰 이미지 (max-height: 150px)
  - "필요시 클릭해서 보세요! (업로드한 이미지) 👆"
  
[채팅 입력창]
  - 입력 필드
  - 전송 버튼
```

## ✅ 제안하는 방식

### 개선안: 헤더 썸네일 개선 + 큰 이미지 제거

### 변경 사항:

#### 1. 헤더 썸네일 개선 (기존 `.imageThumbnail` 유지)
- ✅ 작은 썸네일 이미지 유지 (40x40px)
- ✅ 클릭 시 확대 모달 기능 유지
- ✨ **추가**: 텍스트 안내 추가
  - 썸네일 아래 또는 옆에 "이미지를 보려면 여기를 클릭하세요! 👆" 텍스트 표시

#### 2. 채팅 입력창 근처 큰 이미지 제거
- ❌ `.pageImageNearInput` 영역 전체 제거
- ❌ `.pageImageMini` CSS 제거
- ✅ 채팅 로그 영역이 방해받지 않음

### 새로운 레이아웃:
```
[📷 헤더]
  [프로필 사진] [선생님 이름] 
  [📷 작은 썸네일 (40x40)] 
  "이미지를 보려면 여기를 클릭하세요! 👆" (새로 추가)
  [복습 수업]

[채팅 로그]
  - 메시지들...
  - 메시지들...
  - 메시지들... (이제 방해받지 않음! ✅)
  
[채팅 입력창]
  - 입력 필드
  - 전송 버튼
```

## 🎯 구현 방법

### 방법 1: 썸네일 아래에 텍스트 추가 (세로 배치)
```tsx
{/* 헤더 영역 */}
<div className={styles.chatHeader}>
  <div className={styles.avatarWrap}>
    {/* 프로필 사진과 이름 */}
  </div>
  <div style={{ display: 'flex', alignItems: 'center', gap: '12px' }}>
    {pageImageUrl && (
      <div style={{ display: 'flex', flexDirection: 'column', alignItems: 'center', gap: '4px' }}>
        <div
          className={styles.imageThumbnail}
          onClick={() => {
            // 확대 모달 로직 (기존과 동일)
          }}
          title="페이지 이미지 보기"
        >
          <img src={pageImageUrl} alt="페이지 썸네일" />
          <span>📄</span>
        </div>
        <div style={{ fontSize: '10px', color: '#666', textAlign: 'center', whiteSpace: 'nowrap' }}>
          이미지를 보려면<br />여기를 클릭하세요! 👆
        </div>
      </div>
    )}
    <div className={styles.chatTitle}>복습 수업</div>
  </div>
</div>
```

### 방법 2: 썸네일 옆에 텍스트 추가 (가로 배치) - 공간이 충분할 때
```tsx
{pageImageUrl && (
  <div style={{ display: 'flex', alignItems: 'center', gap: '8px' }}>
    <div className={styles.imageThumbnail} onClick={...}>
      {/* 썸네일 */}
    </div>
    <div style={{ fontSize: '11px', color: '#666', whiteSpace: 'nowrap' }}>
      이미지를 보려면<br />여기를 클릭하세요! 👆
    </div>
  </div>
)}
```

## 📐 CSS 변경사항

### 추가할 CSS (선택 사항):
```css
/* 헤더 썸네일 텍스트 안내 */
.imageThumbnailHint {
  font-size: 10px;
  color: #666;
  text-align: center;
  white-space: nowrap;
  margin-top: 4px;
}

/* 모바일 */

  .imageThumbnailHint {
    font-size: 9px;
  }
}
```

### 제거할 CSS:
- `.pageImageNearInput` 관련 CSS 모두 제거
- `.pageImageMini` 관련 CSS 모두 제거
- `.imageHint` (채팅 입력창 근처 텍스트) CSS 제거

## ✅ 장점

1. ✅ **학습 내용을 가리지 않음**: 채팅 로그 영역이 완전히 자유로움
2. ✅ **항상 접근 가능**: 헤더에 있어서 항상 보임
3. ✅ **시각적 방해 최소화**: 작은 썸네일 (40x40px)만 표시
4. ✅ **명확한 안내**: 텍스트로 "이미지를 보려면 여기를 클릭하세요!" 안내
5. ✅ **기존 기능 유지**: 클릭 시 확대 모달 기능 그대로 유지
